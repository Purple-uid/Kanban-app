# UID_Kanban

Kanban-доска для управления задачами с перетаскиванием карточек, сворачиванием колонок и локальным сохранением.

**Деплой:** [kanban-app-pi-two.vercel.app](https://kanban-app-pi-two.vercel.app/)

---

## Возможности

- **Drag & Drop** — карточки перетаскиваются между колонками с анимацией
- **Сворачивание колонок** — можно свернуть колонку, чтобы освободить место
- **Добавление / удаление колонок и карточек**
- **Автосохранение** — данные сохраняются в localStorage
- **Анимации** — плавные переходы через Motion
